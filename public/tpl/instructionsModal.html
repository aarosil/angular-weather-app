Weather Data Retrieval Instructions

Instructions:

  • Enter a location or click 'Retrieve Location Automatically'
  • Enter two dates up to 9 days apart for time series data
  • -OR- Enter just the start date to view data only for one day
  • -OR- Leave both date fields blank to view the current conditions.
  • Select the data fields (historical reports only)
  • Select the daily averages to calculate (historical reports only)
  • Click 'Load Data'
